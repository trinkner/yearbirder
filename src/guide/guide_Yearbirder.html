What's New in v1.4

  • Add Photos reports new file count — When selecting photos to add, Yearbirder now reports both how many of the selected files are already tracked in your photo catalog and how many are new and will be added. This makes it easy to confirm you have selected the right files before proceeding.
  • Photo deletion updates all open windows — Deleting a photo from disk via the right-click context menu in the Enlargement view now instantly removes that photo from every other open Photos, Enlargement, Manage Photos, and Rename Photos window. Previously, stale thumbnails could remain in other windows until they were manually closed and reopened.
  • Statistics window auto-refreshes after photo changes — The Statistics window now updates automatically when a photo is deleted from disk, and also when the Manage Photos window is closed after saving changes.
  • Default eBird folder prompt only when needed — The offer to set a new default eBird folder now appears only when the file you open is in a different folder from the one already saved in Preferences. Previously it appeared even when opening a file from the same folder.
  • Two new photo charts — The Photos menu now includes New Species Photographed Each Year (a stacked bar chart showing how many species you added to your photographic life list each year) and Photographed Species Growth Over Time (a line curve tracking your total photographed species count over time, with photo thumbnails in the hover tooltip and a click action that opens a Species Gallery for the selected date). See New Species Photographed Each Year and Photographed Species Growth Over Time.
  • Species Gallery gains Slideshow and Show All buttons — The Species Gallery header now includes a Slideshow button (launches a full-screen slideshow of all photos in the current gallery) and a Show All button (opens a Photos window showing every photo matching the gallery's filter).
  • Bubble maps renamed — The two effort bubble maps in the Maps menu have been renamed from "Effort by Time" and "Effort by Checklists" to Bubble by Time and Bubble by Checklists to better reflect that they are bubble maps.
  • eBird Species Code in Rename Photos — The Species Name Format picker in Rename Photos now offers a third option — eBird Species Code — which uses the short eBird alpha code for the species (for example, rufhum for Rufous Hummingbird) instead of the common or scientific name. This is useful for workflows that keep filenames aligned with eBird identifiers.
  • eBird code matching when adding photos — When you add photos from disk and Yearbirder tries to guess which species a photo shows from its filename, it now recognises eBird species codes embedded in the filename (for example, rufhum_20240514.jpg → Rufous Hummingbird) as its first matching strategy — before the existing BBL code and word-token checks.

1. Quick Start

This section gets you from a fresh install to your first results in five minutes.

Step 1 — Download your eBird data

With a browser such as Chrome, Safari, or Edge, go to https://ebird.org/downloadMyData, and choose Request My Observations. eBird will email you a link to a .zip file containing your complete sightings history. Open the email, click the link, and save the file somewhere easy to find. Yearbirder can open the .zip file directly — there is no need to unzip it first. By default, most browsers will save the file in your Downloads folder.

Step 2 — Open your data in Yearbirder

Launch Yearbirder. Click the Open button on the toolbar (or choose File → Open). Navigate to the .zip file you downloaded and click Open. Yearbirder loads your data and displays list of the species in the file.

Step 3 — Generate your first list

Browse through the options on the filter on the left side of the screen. Set a few filter options, like county or family, and then click the Species button on the toolbar. A new window opens showing the species you have recorded, sorted taxonomically, along with the first date, most recent date, and checklist count for each. You can click on the column headers to sort the columns. You can also right-click on any information in the list to send that information to the filter.

Step 4 — Explore further

  • Click Map to see all your sighting locations plotted on an interactive map.
  • Click Date Totals to see which years, months, and individual dates you recorded the most species.
  • Click Photos (after setting up a photo catalog — see Section 17) to browse your bird photos alongside their sighting data.

You are up and running. Everything else in this guide describes these features in detail.

Note that if some checklists include exotic species, the totals that Yearbirder calculates may not match the totals reported by eBird. The data file you download from eBird does not specify which sightings are exotics, so Yearbirder cannot exclude them from its totals.

2. The Main Window

Yearbirder allows many report, map, and graph windows to be open at the same time. The main window contains:

  • A menu bar across the top with menus for File, Filter, Lists, Reports, Graphs, Maps, Photos, Windows, and User Guide.
  • A toolbar just below the menu bar with quick-access buttons for the most common actions.
  • A Sighting Filter panel docked on the left side.
  • A Photo Filter panel docked on the right side.
  • A large central area where all open windows (lists, reports, maps, photos) are displayed.
  • A status bar at the very bottom showing the current data file and record counts.

Working with multiple windows

Each window's title bar shows a descriptive label that reflects the active filter — for example, "Species: Warbler; New York; 2024 (42 Species)".

3. Loading Your eBird Data

Opening a data file

Choose File → Open eBird data file… or click the Open toolbar button. Yearbirder accepts the standard eBird download data file (typically named something like MyEBirdData.zip).

Loading time depends on the size of your dataset. A typical dataset of 10,000–50,000 sightings loads in a few seconds. Once loaded, the status bar shows the total number of sightings and species.

To avoid having to open your eBird data file manually every time you use Yearbirder, save it as a preference (see Section 27 — Saving Preferences) so that Yearbirder opens it automatically at startup.

Closing a data file

Choose File → Close eBird data file to unload the current dataset. Any open windows that depend on the data will close automatically.

What data is loaded

Yearbirder reads the following fields from your eBird data file:

  • Species common name and scientific name
  • Taxonomic order, family, and order
  • Observation date and time
  • Country, state/province, county, and location name
  • Location coordinates (latitude/longitude)
  • Checklist ID
  • Number of individuals counted

4. The Sighting Filter

The Sighting Filter panel is the primary way to focus every report and list on the data you care about. It is docked on the left side of the main window and always visible. All open windows update immediately whenever you change any filter setting.

Location filters

The location filters are hierarchical — selecting a more specific level automatically constrains the broader levels. You can filter at any one of these levels:

FilterDescription
RegionA broad geographic region (e.g., North America)
CountryA specific country (e.g., United States)
StateA state or province
CountyA county or sub-region within a state
LocationA specific named eBird location

Select **All** in any drop-down to clear that level's filter.

Taxonomic filters

FilterDescription
OrderBird order (e.g., Passeriformes)
FamilyBird family (e.g., Parulidae – Wood-Warblers)
SpeciesA single species name
Name SearchFree-text search; matches any part of a species' common name

The taxonomic filters also follow a hierarchy for the purposes of the window title: if a Species is selected, Family and Order are not shown in the title (they are implied). If only a Family is selected, Order is not shown. Only the most specific level selected is displayed.

Date filters

FilterDescription
Date OptionsQuick presets: No date filter, This year, Last year, etc.
Start DateBeginning of a custom date range (yyyy-mm-dd)
End DateEnd of a custom date range (yyyy-mm-dd)

Seasonal range filters

Seasonal filters let you examine a recurring window within each year regardless of the year itself — useful for studying migration patterns.

FilterDescription
Seasonal Range OptionsQuick presets for common seasonal windows
Start Seasonal RangeMonth and day marking the start of the seasonal window
End Seasonal RangeMonth and day marking the end of the seasonal window

For example, setting a seasonal range of May 1 – May 31 shows all May sightings across all years.

Clearing filters

  • Click Clear all filters on the toolbar to reset every filter at once.
  • Individual filters can be cleared by selecting the **All** option in their drop-down.

It filters photo-related results in addition to the Sighting Filter — only photos that match both the Sighting Filter and the Photo Filter are shown.

6. Species List

Open with Lists → Species or the Species toolbar button.

The Species List shows every bird species (and other taxa) you have recorded that matches the current filter. Each row shows:

ColumnDescription
TaxTaxonomic sort order number
SpeciesCommon name (click to open the Individual Species window)
FirstDate of your first sighting of this species
LastDate of your most recent sighting
ChecklistsNumber of checklists on which this species appears
% of ChecklistsThe percentage of your total checklists that include this species

Species that are not full species (slash species, spuhs, hybrids) are listed separately and marked distinctly.

The window title shows the total count — for example, "Species: New York (42 Species)".

Clicking a species

Click any species name to open an Individual Species window (see Section 24) showing the full history of your sightings of that species.

Right-click menu

Right-clicking a row in the species table offers options to set the species as the active filter, open associated web pages, or open the individual species window.

9. Families Window

Open with Lists → Families or the Families toolbar button.

The Families window has two panes:

  • Left pane: A scrolling list of every bird family represented in your sightings under the current filter.
  • Right pane: The species from the selected family, along with sighting statistics.

Click a family name in the left pane to populate the right pane. This is an efficient way to browse your sightings by taxonomic family — for example, quickly seeing all your warbler or shorebird records.

The window title shows the number of families, e.g., "Families: New York (12 Families)".

10. Date Totals

Open with Reports → Date Totals or the Date Totals toolbar button.

The Date Totals window presents three side-by-side tables:

Year Totals

One row per year, showing how many species and checklists you recorded in that year.

Month Totals

One row per calendar month (January through December), showing totals across all years.

Date Totals

One row per individual date on which you submitted a checklist, showing species and checklist counts for that date.

All three tables are sortable by any column. This report is useful for identifying your most productive years, your best months for birding, and your record single-day species counts.

The window title shows the total number of unique dates, e.g., "Date Totals: All species, locations, and dates (365 Dates)".

11. Location Totals

Open with Reports → Location Totals or the Location Totals toolbar button.

Location Totals presents five tables on separate tabs:

TabDescription
Region TotalsSpecies counts by broad geographic region
Country TotalsSpecies counts by country
State TotalsSpecies counts by state/province
County TotalsSpecies counts by county
Location TotalsSpecies counts by individual named location

Each table shows rank, name, species count, and checklist count. Sorted by rank (most species first) by default.

12. Big Report

Open with Reports → Big Report or the Big Report toolbar button.

The Big Report is the most comprehensive single report in Yearbirder. It combines multiple dimensions of your data into one window with multiple tabs:

  • Species — Full species list with counts, first and last dates.
  • Dates — Date-by-date breakdown of species seen.
  • Locations — Location-by-location breakdown.
  • New Species — Species seen for the first time at each location.
  • Checklists — All checklists in the filtered set.

The window title includes a species count, e.g., "Big Report: New York (87 Species)".

13. Compare Lists

Open with Reports → Compare Lists or by clicking the Compare Lists button in the toolbar.

Compare Lists lets you select two separate lists and compare their species. This is useful for questions like:

  • "Which species did I see in 2023 that I did not see in 2024?"
  • "Which species have I seen at location A but not at location B?"

To compare lists:

  1. Use the main window's filter and the Species button on the toolbar to create at least two list windows.
  2. Click the Compare Lists button on the toolbar to open the Compare Lists window.
  3. A card is shown for each open species list. Click one card to select it as List A (highlighted in blue). Click a second card to select it as List B (highlighted in orange) — the comparison runs immediately.

To change a selection, click the highlighted card again to deselect it, then click a different card. Clicking an unselected card when both slots are already filled automatically replaces List B.

Results are displayed in three columns: species unique to List A on the left, species on both lists in the centre, and species unique to List B on the right. Double-click any species name to open its Individual Species window.

14. Graphs

Open graphs from the Graphs menu. A variety of chart types are available, each responding to the current Sighting Filter. All graphs display a hover tooltip when you move the cursor over a data point or bar — the tooltip shows relevant details such as date, species count, or location. Clicking on most data points opens a Species List window filtered to that point's context.

Yearbirder uses a consistent colour scheme across all graphs: charts based on sighting data use a blue theme, while charts based on photograph data (opened from the Photos menu, or any graph produced when the Photo Filter is active) use a yellow theme.

Bar Graph

Open with Graphs → Bar Graph.

Shows species counts aggregated over time, with four granularity options selectable via radio buttons at the top of the window:

  • By Year — one bar per year; the default when your data spans multiple years.
  • By Month — one bar per calendar month, aggregated across all years. Shows which months are most productive regardless of year.
  • By Month-Year — one bar per calendar month within each year (e.g., Jan-2022, Feb-2022, …). Useful for fine-grained temporal analysis. When there are more than 50 bars, x-axis labels are shown for every nth bar only to prevent crowding; hover tooltips remain available for all bars.
  • By Day — one bar per individual date. Useful when the filter covers a narrow date range. Disabled if there are more than 1,000 days in the filtered data. When there are more than 50 bars, x-axis labels are thinned automatically; hover tooltips remain available for all bars.

Click any bar to open a Species List for that time period.

Cumulative Species Curve

Open with Graphs → Cumulative Species Curve.

A line chart showing how your total species count has grown over time as each new species was first recorded. The x-axis is date and the y-axis is the running cumulative total. The final point shows the total number of species seen under the current filter.

This chart is most interesting with no date filter set — it shows your life-list growth curve from your earliest eBird records to the present. With a location filter applied, it shows how quickly you added new species at that location.

Hover over the line to see the cumulative count and date at any point. Click to open a Species List of all species seen up to and including that date.

Species Heatmap

Open with Graphs → Species Heatmap.

A grid showing species counts for every month of every year in your data. Rows are years (most recent at the top) and columns are months (January through December). Cell colour ranges from light blue (low count) to dark blue (high count); cells with no data are shown in grey. When the grid is 15 years or fewer, each cell is also labelled with its species count.

The heatmap immediately reveals seasonal patterns — which months of each year were most productive — and gaps in your birding record.

Hover over any cell to see the month, year, and species count. Click a cell to open a Species List for that month and year.

Species Accumulation

Open with Graphs → Species Accumulation.

A stacked bar chart showing, for each year, how many species you recorded that were new to you under the current filter (bright blue, labelled with a + count) versus how many were species you had already seen in a previous year (muted blue). Together the two segments show your total species count for that year.

This graph answers questions such as: "How many new species did I add to my county list in 2023?" or "Has my backyard species accumulation slowed down in recent years?"

Hover over a bar to see the year, total count, and new-species count. Click a bar to open a Species List for that year.

Top Locations

Open with Graphs → Top Locations.

A horizontal bar chart showing your top locations by species count (up to 20 locations). Each bar is split into two segments:

  • Muted blue — species you had already recorded elsewhere before visiting this location.
  • Bright blue — species you recorded at this location for the first time ever in your database (life birds).

The total species count is labelled at the right end of each bar. This chart shows not only which locations have been most productive, but which locations have contributed the most unique species to your life list.

Hover over a bar to see the full location name, total species count, and life bird count. Click a bar to open a Species List filtered to that location.

Checklist Scatter

Open with Graphs → Checklist Scatter.

A scatter plot showing every checklist matching the current filter as a dot, with duration in minutes on the x-axis and species count on the y-axis. Dots are coloured by season:

ColourSeasonMonths
Steel blueWinterDecember 20 – March 19
GreenSpringMarch 20 – June 19
GoldenSummerJune 20 – September 19
OrangeFallSeptember 20 – December 19

Incidental observations (checklists with no recorded duration) are excluded; a note at the bottom right of the chart shows how many were omitted.

Important: the species count on the y-axis reflects only species matching the active filter, not all species on the checklist. For example, if you filter by Tyrannidae, a checklist on which you recorded 3 flycatchers out of 40 total species will appear at y = 3.

Hover over a dot to see the date, location, duration, and species count. Click a dot to open a Species List for that checklist.

Phenology Chart

Open with Graphs → Phenology Chart.

A dot chart designed to reveal the seasonal timing of species occurrence. Each dot represents one date on which a species (or species matching the filter) was recorded, plotted by day of year on the x-axis and year on the y-axis. Dots are coloured by season using the same palette as the Checklist Scatter. If the same species was recorded at multiple locations on the same day, only one dot is plotted for that date (the tooltip lists all the locations).

This chart is most powerful when a species, family, or order filter is set. For a single species it immediately shows:

  • Typical arrival and departure dates
  • Whether the species is a year-round resident or a seasonal visitor
  • Whether the timing has shifted over the years
  • Unusually early or late individual records

If no species, family, or order filter is set, Yearbirder will warn you before generating the chart, since plotting all species together produces a very cluttered result.

Hover over a dot to see the exact date, the location or locations seen that day, and the species recorded at that location on that date. Click a dot to open a Species List for that date.

First of Year Chart

Open with Graphs → First of Year Chart.

A dot chart plotting the first sighting of each species in each year. The x-axis shows day of year (January through December) and the y-axis shows year. Each dot represents one or more species whose first record of the year fell on the same date; hovering reveals the species list for that dot. Dots are coloured by season using the same palette as the Checklist Scatter.

This chart is ideal for tracking arrival timing — when do your warblers return each spring? Has arrival crept earlier over the years? Apply a Family or species filter before opening the chart for the most focused view.

Hover over a dot to see the date, the location of the first sighting, and the species recorded for the first time that year. Click a dot to open a Species List for that date.

Last of Year Chart

Open with Graphs → Last of Year Chart.

The mirror image of the First of Year Chart: plots the last sighting of each species in each year, using the same axes and colour scheme. Useful for tracking departure timing and lingering late records.

Hover over a dot to see the date, the location of the last sighting, and the species recorded for the last time that year. Click a dot to open a Species List for that date.

YTD Bar Charts

Open from Graphs → Bar Charts. Four year-to-date charts compare your progress through the current calendar date against your full-year totals for every year in your data.

Each chart displays a horizontal bar for each year. The blue segment shows the year-to-date total (up to today's month and day) and the muted blue extension shows how much more was recorded after the same date in that year. The current year's bar is highlighted. All charts respond to the Sighting Filter.

Hover over a bar to see the year, YTD total, full-year total, and a list of the most recently added items. Click a bar to open a filtered window showing the records for that year through today's date.

  • YTD Report by Species — counts distinct species. Tooltip shows the species most recently added to your YTD list.
  • YTD Report by Locations — counts distinct locations visited. Tooltip shows the locations most recently added.
  • YTD Report by Checklists — counts checklists submitted. Tooltip shows the most recent checklists (date and location).

Pie Charts

Open with Graphs → Pie Charts. Two pie chart variants are available, selectable via radio buttons at the top of the window:

  • Pie Chart by Species — each wedge represents a bird family (or order), sized by the number of species you have recorded in it under the current filter.
  • Pie Chart by Individual Tallies — each wedge represents a family (or order), sized by the total number of individual birds tallied. Unknown counts ("X" in eBird) are counted as 1.

Toggle between Families and Orders using the radio buttons below the chart title. Wedges smaller than 0.5% of the total are unlabelled to avoid clutter; percentage labels appear only on wedges of 2% or larger.

Hover over a wedge to see the family or order name, its count, and the species it contains. Click a wedge to open a Species List filtered to that family or order.

15. Maps

Yearbirder's Maps menu offers a range of interactive map types. All maps are pan-and-zoom capable using your mouse or trackpad, respond to the Sighting Filter, and open just like any other report. Hovering over a marker or bubble shows a tooltip; clicking most markers opens the Location window for that site.

Location Map

Open with the Map toolbar button or Map → Map.

Plots every location where you have recorded sightings under the current filter as a circle marker, sized by species count. Hover over a marker to see the location name and species list. Click a marker to open the Location window for that site.

Animated Lifer Map

Open with Maps → Animated Lifer Map.

Animates your life list building geographically over time. Each dot represents a lifer — the first time you recorded a species anywhere in your database — placed at the sighting's GPS coordinates and revealed in chronological order. Dots are coloured on a cream-to-orange gradient from earliest to most recent lifer; the newest dot flashes blue before settling to its gradient colour when the next dot appears.

A control bar at the bottom of the map provides:

  • ⏮ Restart — jump back to the beginning.
  • ▶ / ⏸ Play/Pause — start or pause the animation.
  • Scrubber — drag to jump to any point in the sequence.
  • Speed slider (🐢 – 🐇) — adjust animation speed from leisurely to rapid.
  • Lifer info line — shows the species name, location, and date of the most recently revealed dot.

A tooltip showing the species name, lifer number, date, and location follows the animation automatically as each new dot appears. Hover any dot at any time to see its details; click a dot to open a popup with the same information.

Animated First Sightings Map

Open with Maps → Animated First Sightings.

Animates the accumulation of new species under the current filter, revealing each species at the location where you first recorded it — in chronological order. Unlike the Animated Lifer Map, which always shows absolute life firsts, this map respects the active filter: apply a county filter to watch your county list grow, apply a year filter to replay a single year's additions, or apply any combination of region, family, or date filters to explore a specific context.

Dots are coloured on a cream-to-orange gradient — pale dots are earlier firsts, deep orange dots are more recent. The newest dot appears in bright red and settles to its gradient colour when the next dot arrives. A tooltip showing the species name, number, date, and location follows the animation automatically; hover any dot at any time to see its details.

The control bar is identical to the Animated Lifer Map:

  • ⏮ Restart — jump back to the beginning.
  • ▶ / ⏸ Play/Pause — start or pause the animation.
  • Scrubber — drag to jump to any point in the sequence; all dots up to that point are shown immediately.
  • Speed slider (🐢 – 🐇) — adjust animation speed from leisurely to rapid.
  • Species info line — shows the species name, location, and date of the most recently revealed dot.

Bubble by Time

Open with Maps → Bubble by Time.

A bubble map where each bubble represents a checklist location. Bubble area is proportional to the total cumulative time you have spent birding at that location (in minutes, summed across all matching checklists). Locations where you have invested the most time appear as the largest bubbles.

Hover over a bubble to see the location name, total time, and checklist count. If some checklists at a location have no recorded duration, the tooltip notes how many are untimed. Click a bubble to open the Location window for that site.

Bubble by Checklists

Open with Maps → Bubble by Checklists.

The same bubble map concept, but bubble area is proportional to the number of checklists submitted at each location rather than total time. Useful for seeing where you bird most frequently regardless of duration.

Hover over a bubble to see the location name, checklist count, and total time. Click a bubble to open the Location window for that site.

Species Total Map

Open with Maps → Species Total Map.

A bubble map where bubble area is proportional to the number of species recorded at each location under the current filter. Slashes, spuhs, and hybrids are excluded from the species count using the same rules as the Species List. Apply a taxonomic filter to compare how many warblers — or any other group — you have found at each of your locations.

Hover over a bubble to see the location name and species count. Click a bubble to open the Location window for that site.

Individuals Total Map

Open with Maps → Individuals Total Map.

A bubble map where bubble area is proportional to the total number of individual birds tallied at each location. Sightings recorded with an unknown count ("X" in eBird) contribute 1 to the total. The tooltip notes how many checklists contain at least one uncounted species.

Hover over a bubble to see the location name, individual count, and any caveat about uncounted entries. Click a bubble to open the Location window for that site.

The Geolocated Photos map has moved to the Photos menu. See Section 18.

16. Choropleth Maps

Choropleth maps are accessed via Maps → Choropleths. Six geographic regions are available, each with two modes selectable from the By Species and By Checklists submenus.

All choropleth maps respond to the Sighting Filter — for example, set a Family filter to see which states you have recorded warblers in. Clicking a shaded region opens a Species List (By Species mode) or a Checklists List (By Checklists mode) filtered to that region.

By Species mode

Each region is shaded according to the number of distinct species you have recorded there under the current filter. Darker shading = more species. Hover over a region to see its name, species count, and a list of the most recently recorded species.

By Checklists mode

Each region is shaded according to the number of checklists you have submitted there. Hover over a region to see its name, checklist count, and the dates of your most recent visits (newest first).

17. Adding Photos to Yearbirder

Yearbirder uses a photo catalog to associate photos on your hard drive with your eBird sightings. Once photos are associated, Yearbirder can show you all the photos you have taken of a species, or the photos from a particular day or location. The catalog stores file paths, dates, species names, ratings, and camera information. Think of it as a photo library that sits alongside your eBird data.

Adding photos to Yearbirder

  1. Choose Photos → Add Photos.
  2. A file dialog opens. Navigate to the folder containing your bird photos and select one or more JPEG files. You can select multiple photos at once — there is no need to add them one at a time:
    • Mac: hold Command (⌘) and click to select individual photos, or hold Shift and click to select a contiguous range.
    • Windows: hold Ctrl and click to select individual photos, or hold Shift and click to select a contiguous range.
  3. For each photo, Yearbirder asks you to confirm the associated bird species and checklist. To make this easier, it uses each photo's filename and EXIF date/time to suggest the most likely species — so in many cases you only need to confirm the suggestion. Yearbirder recognises eBird species codes in filenames (e.g. rufhum_20240514.jpg), BBL four-letter codes, and common-name words. If the correct sighting does not appear as an option, make sure that sighting is logged in eBird; you cannot associate a photo with a sighting that is not in your data.
  4. Once you have associated photos, choose Photos → Save Photo Catalog to save your catalog file. Remember where you save it — you will open it again the next time you use Yearbirder.

Opening an existing photo catalog

Choose Photos → Open Photo Catalog and navigate to your saved catalog file. Yearbirder loads all photo associations and EXIF data. The Photo Filter panel becomes active.

To avoid opening your photo catalog manually every time, save its path as a preference (see Section 27 — Saving Preferences) so that Yearbirder opens it automatically at startup.

Saving changes

After adding, editing, or removing photo associations, choose Photos → Save Photo Catalog to write changes back to the file. Yearbirder will warn you if you try to close with unsaved changes.

Closing the photo catalog

Choose Photos → Close Photo Catalog. This unloads all photo data from memory. The Photo Filter panel becomes inactive.

Adding more photos

With a photo catalog already loaded, choose Photos → Add Photos to import additional photos. The new associations are merged into the existing catalog; save afterwards. After you select files, Yearbirder reports how many are already tracked and how many are new — so you can confirm you have selected the right files before any associations are created.

Managing photos

Choose File → Manage Photos… to open the Manage Photos window, where you can view and edit the details of individual photos that match the current filter — for example, to reassign a photo to a different species or correct a date.

Updating EXIF data

If you have moved or re-exported your photo files, choose Photos → Update EXIF Data for All Photos. Yearbirder re-reads the EXIF metadata from every file referenced in the catalog and updates its records. This is also useful after photo editing software writes new EXIF tags.

Cleaning up missing references

If photos have been moved or deleted from your hard drive, Yearbirder may show broken references. Use Photos → Remove Missing Photo References to scan for and remove any associations pointing to files that no longer exist.

What the photo catalog contains

The catalog stores one record per photo, including:

  • File path to the JPEG
  • Associated species common name
  • Associated sighting date
  • Your personal rating
  • Camera model (from EXIF)
  • Lens model (from EXIF)
  • Shutter speed (from EXIF)
  • Aperture (from EXIF)
  • Focal length (from EXIF)
  • ISO (from EXIF)

You can open the catalog file in a text editor or spreadsheet to inspect values, but be careful to preserve its structure.

18. Viewing Photos

Yearbirder can associate your personal bird photos with your eBird sightings. This unlocks several photo views and lets you filter by camera, lens, and exposure settings using the Photo Filter panel on the right of the screen.

Overview of the photo system

  • Your photos remain in their original folders on your computer. Yearbirder stores references to them, not copies.
  • Photo associations and metadata are stored in a photo catalog (a file you create and save using Yearbirder). See Section 17.
  • Yearbirder reads EXIF data from each JPEG file automatically: camera model, lens, shutter speed, aperture, focal length, and ISO.

Browse Photos

Open with Photos → Browse Photos, the Photos toolbar button, or Lists → Photos. Shows thumbnails of every photo matching the current Sighting Filter and Photo Filter.

Each thumbnail shows the photo, species name, date, and your rating. Click any thumbnail to open the Enlargement view.

A Slideshow button appears in the top-right corner of the Photos window. Click it to launch a full-screen slideshow of the photos currently shown — the same configuration dialog appears first, letting you choose sort order, speed, and title-bar options. See Slideshow for details.

Sorting photos

Use the radio buttons at the top of the Photos window to sort by:

  • Taxonomy — groups photos by species in taxonomic order
  • Date — chronological order
  • Rating — highest rated first
  • Alphabetical — by species common name

Photo Filter

Use the Photo Filter panel (right side of the main window) to narrow results by camera, lens, exposure settings, or rating. See Section 5 for the full list of filter options.

Enlargement view

Click any thumbnail in the Photos window to open the Enlargement view, which shows the photo at full size with pan and zoom. From the Enlargement view you can:

  • Navigate between photos using the previous/next arrows. Pressing F10 or using the right-click context menu enters full-screen mode; the left and right arrow keys navigate between photos with a smooth crossfade transition.
  • Set or change the rating for the photo.
  • View EXIF data — camera, lens, shutter speed, aperture, focal length, ISO.
  • Detach a photo from its sighting (removes the association without deleting the file).
  • Delete a photo from disk — permanently removes the file from your drive and its entry from the photo catalog. All other open Photos, Enlargement, Manage Photos, and Rename Photos windows update immediately to reflect the deletion.

Geolocated Photos

Open with Photos → Geolocated Photos. Available only when a photo catalog is loaded.

Plots your geotagged photos on an interactive clustered marker map. Markers group together at lower zoom levels and spread apart as you zoom in. Hovering over a marker shows a thumbnail preview of the photo along with the species name, date, and location. Click a marker to open the Enlargement view for that photo. Multiple photos taken at exactly the same GPS coordinates are automatically fanned into a spiral so each remains individually clickable.

Animated Sequence Map

Open with Photos → Animated Sequence Map. Available only when a photo catalog is loaded.

Animates your geotagged photos in chronological order on an interactive map, showing where and when each photo was taken. Each photo appears as a callout bubble whose triangular tail points to the exact GPS location, so the underlying map detail remains visible. The bubble displays the photo thumbnail, species name, date, and location.

As the animation progresses, a small blue dot is permanently placed at each visited location, building up a breadcrumb trail of your photographic journey.

Playback is controlled by the bar at the bottom of the map:

  • Play / Pause — start or pause the animation.
  • Restart — return to the first photo and clear the breadcrumb trail.
  • Scrubber — drag to jump directly to any photo in the sequence.
  • Speed slider — drag towards the rabbit for faster playback (minimum 0.25 seconds per photo) or towards the turtle for slower playback (maximum 5 seconds per photo).

The caption bar below the controls shows the species name, location, and date of the currently displayed photo. Click the photo bubble at any time to open the Enlargement view for that photo.

Slideshow

Open with Photos → Slideshow. Available only when a photo catalog is loaded.

Displays your photos as a full-screen slideshow with smooth crossfade transitions between images. Before the slideshow begins, a dialog lets you configure it:

  • Sort Order — choose how photos are sequenced:
    • Random — photos appear in a randomised order (default).
    • Taxonomic — grouped by species in taxonomic order.
    • Alphabetic — sorted by species name A–Z.
    • By rating (highest first) — your best-rated photos appear first.
    • Chronological — sorted by date and time taken.
    • By location — grouped alphabetically by location name.
    • By seasonal date (month / day) — sorted by calendar date, ignoring year, so all January photos appear first regardless of which year they were taken.
  • Seconds per photo — how long each photo is displayed before the next crossfade begins (1–60 seconds).
  • Show title bar — when checked, a dark panel at the bottom of the screen shows the species name, date, location, and a photo counter. Uncheck for a clean full-screen view with no overlay.

Press any key or move the mouse to exit the slideshow at any time. The cursor is hidden automatically while the slideshow is running.

YTD Report by Photographs

Open with Photos → YTD Report by Photographs. Available only when a photo catalog is loaded.

A horizontal bar chart comparing your year-to-date photo count against your full-year photo totals for every year in your data. The yellow segment shows photos taken up to today's date in each year; the muted amber extension shows photos taken after the same date. The current year's bar is highlighted.

Hover over a bar to see the year, YTD photo count, full-year total, and the species you most recently photographed. Click a bar to open a Photos window showing all photos from that year through today's date.

Families & Orders by Photos

Open with Photos → Families & Orders by Photos. Available only when a photo catalog is loaded.

A pie chart where each wedge represents a bird family (or order), sized by the total number of photos you have of species in that group under the current filter — not the number of species or individuals, but the literal photo count. Wedges are rendered in a yellow colour palette to distinguish this chart from the sighting-data pie charts.

Toggle between Families and Orders using the radio buttons. Hover over a wedge to see the group name, photo count, percentage of total, and the species you have photographed most within that group (ranked by photo count). Click a wedge to open a Photos window filtered to that family or order.

Total Photos

Open with Photos → Total Photos. Available only when a photo catalog is loaded.

A yellow-themed bar chart showing your total photo count per time period. Use the four radio buttons at the top to switch between By Year, By Month, By Month-Year, and By Day views — the same granularity controls as the Total Species bar chart.

Hover over a bar to see the period label, total photo count, and the species you photographed most during that period (listed in descending photo-count order, with the count per species shown in parentheses). Click a bar to open a Photos window showing all photos from that time period.

New Species Photographed Each Year

Open with Photos → New Species Photographed Each Year. Available only when a photo catalog is loaded.

A yellow-themed stacked bar chart showing how many species you added to your photographic collection each year. Each bar is split into two segments:

  • Bright amber — species photographed for the first time ever under the current filter in that year (new to your photo life list).
  • Muted amber — species you had already photographed in a previous year (repeat species).

This chart answers questions such as: "How many new species did I photograph for the first time in 2023?" or "Has my rate of adding new species to my photo list been slowing down?"

Hover over a bar segment to see the year, the total species count, and a list of the new species first photographed in that year. Click a bar to open a Species Gallery for that year.

Photographed Species Growth Over Time

Open with Photos → Photographed Species Growth Over Time. Available only when a photo catalog is loaded.

A yellow-themed line curve showing how your total number of photographed species has grown over time. The x-axis is date (using the date each species was first photographed) and the y-axis is the running cumulative total of species you have ever photographed under the current filter.

This is the photographic equivalent of the Cumulative Species Curve — the final point tells you the total number of distinct species you have photographed.

Hover over the curve to see the date, cumulative total, and a tooltip that shows a photo thumbnail for each new species first photographed on that date (up to six thumbnails, stacked vertically; the thumbnail is the highest-rated photo of that species taken on that day). Click a point to open a Species Gallery showing only the species first photographed on that date, filtered to photos taken on that day.

19. Working with the Enlargement View

The Enlargement view opens whenever you click a photo thumbnail anywhere in Yearbirder — in the Photos window, the Species Gallery, the Individual Species window, the Geolocated Photos map, or the Animated Sequence Map. It shows your photo at full resolution against a dark background and doubles as a photo management tool: you can navigate your collection, rate images, inspect EXIF data, and perform file operations all without leaving the view.

Navigating between photos

Use the left and right arrow keys (or Page Up / Page Down) to step through the photos in your current set. The Space bar advances to the next photo; Backspace goes back. You can also right-click and choose Next photo or Previous photo from the context menu.

Zooming and panning

Scroll the mouse wheel up to zoom in and down to zoom out. Once zoomed in, click and drag to pan around the photo. Press F (or right-click → Fit to window) at any time to snap the photo back to fit the window.

Full-screen mode

Press F10 (or right-click → Full screen) to expand the Enlargement view to fill the entire screen, hiding the Yearbirder interface. Arrow keys still navigate between photos in full-screen mode, and you can still zoom and pan. Press F10 again or Esc to return to the windowed view.

The details pane

Press F9 (or right-click → Show details) to open the details pane along the bottom of the Enlargement view. The pane shows:

  • The species common name and scientific name
  • EXIF data: camera model, lens, shutter speed, aperture, focal length, and ISO
  • Five clickable star buttons for rating the photo

Press F9 again to close the pane and reclaim the screen area for the photo.

Rating photos

You can assign a 1–5 star rating from the Enlargement view in three ways:

  • Keyboard — press 1 through 5 to assign the corresponding rating instantly. Press 0 to clear the rating. These shortcuts work whether or not the details pane is open.
  • Star buttons — open the details pane (F9) and click one of the five star buttons.
  • Context menu — right-click and choose Rate 1 star through Rate 5 stars.

Hiding the cursor

Press F7 (or right-click → Hide cursor) to make the cursor invisible, giving you a clean view of the photo without anything obscuring the detail you are trying to evaluate. Press F7 again (or right-click → Show cursor) to restore it.

Detaching and deleting photos

Right-click on the photo to access two file-management actions:

  • Detach photo from Yearbirder — removes the photo's association with its sighting and drops it from your catalog, but leaves the file on disk untouched. Use this when you want Yearbirder to stop tracking a photo without deleting it.
  • Delete photo from file system — permanently deletes the file from your drive and removes it from your catalog. All open Photos, Enlargement, Manage Photos, and Rename Photos windows refresh immediately. This action cannot be undone.

20. Rating Your Photos

Ratings are a 1–5 scale (or 0 for unrated) that you assign to each photo. They are stored in your photo catalog and can be used to filter photos in any Yearbirder photo view using the Rating Range control in the Photo Filter, and to sort photos by quality in the Photos gallery. You can assign or change a rating in any of the following ways:

On import

When you add photos via Photos → Add Photos, the confirmation dialog for each photo includes a rating field. You can assign an initial rating before the photo is saved to your catalog — useful for flagging your best shots immediately after a field outing.

Manage Photos window

Choose File → Manage Photos… to open the Manage Photos window. It shows all photos matching the current filter, one per row, and lets you set or change the rating for each photo individually. Use the Sighting Filter to narrow the list to a specific species, date range, or location before opening it.

Right-click context menu in the Enlargement view

Right-click on a photo while viewing it in the Enlargement view and choose Rate 1 star through Rate 5 stars from the context menu. The rating is applied immediately and the details pane updates to reflect the change.

Keyboard shortcuts in the Enlargement view

While viewing a photo in the Enlargement view, press 1 through 5 to assign the corresponding rating without using the mouse. Press 0 to clear the rating. These shortcuts work whether the details pane is visible or not.

Star buttons in the details pane

If the details pane is visible (toggle with F9), five star buttons appear at the bottom of the pane. Click any star to assign that rating. Clicking the 1-star button a second time clears the rating back to unrated.

21. Managing Your Photos

Choose File → Manage Photos… to open the Manage Photos window. It shows every photo matching the current Sighting Filter — one photo per row — with a large thumbnail on the left and a set of dropdown controls on the right. Use these controls to reassign a photo to a different checklist, correct a mis-identified species, or change its rating.

The dropdown controls

Each photo row has five dropdowns that identify exactly which eBird checklist sighting the photo is attached to:

  • Location — the eBird location associated with the photo. Change this to redirect the photo to a checklist at a different location.
  • Date — the checklist date at the selected location. The available dates update automatically when you change the location.
  • Time — the checklist start time for the selected location and date. The available times update automatically when you change the date.
  • Species — the species on the selected checklist to which this photo is attached. The available species update automatically when you change the time. To remove a photo from the catalog without deleting the file from disk, choose **Detach Photo** from this dropdown.
  • Rating — the photo's quality rating (Not Rated, or 1–5). See Rating Your Photos for other ways to set ratings.

Any dropdown that differs from the photo's current saved value is highlighted. Click the Reset button on a row to revert all of that photo's dropdowns to their original values.

Saving changes

Click OK to apply and save all changes across every photo shown in the window. Changes are written immediately to your photo catalog. Click Cancel to close the window without saving.

22. Rename Photos

Open with Photos → Rename Photos. Available once a photo catalog is loaded.

Rename Photos lets you batch-rename your photo files on disk using a structured name you compose from up to four information slots. The new names are built from the data Yearbirder already knows about each photo — its species, date, time, and location — so the resulting filenames are descriptive and consistently formatted. After renaming, Yearbirder automatically updates all references in your photo catalog.

Renaming is permanent and cannot be undone. Back up your photos before proceeding.

Name Format

The top section of the window lets you build the filename template. Choose up to four slots, each of which contributes one piece of information to the name. Slots are joined by underscores. Any slot can be set to (omit) to leave it out.

Slot optionWhat it produces
Species NameThe species common or scientific name (your choice)
DateThe date the photo was taken, in your chosen format
TimeThe time the photo was taken, in your chosen format
LocationThe eBird location name associated with the sighting
(omit)This slot is skipped

The default order is Date → Time → Species Name → Location, which produces filenames like 2024-05-14_07-23-45_Black-throatedBlueWarbler_CentralPark.jpg.

Additional format controls appear below the slots:

  • Date Format — YYYY-MM-DD, YYYYMMDD, or (omit).
  • Time Format — HH-MM-SS, HHMMSS, HH-MM-SSt (with tenths digit), HHMMSSt, or (omit). Time is read from the photo's EXIF data.
  • Species Name Format — Common Name, Scientific Name, or eBird Species Code. The eBird Species Code option uses the short eBird alpha code (for example, rufhum for Rufous Hummingbird, amecro for American Crow), which is useful for workflows that keep filenames aligned with eBird identifiers. Requires an eBird taxonomy file to be loaded at startup.

A Sample line below the format controls shows a live preview of what the first photo's new name would look like as you adjust your settings.

Options

  • Remove spaces — strips spaces from species and location name components so the filename contains no whitespace (on by default).
  • Use checklist time for photos without a time — when a photo has no EXIF time, falls back to the eBird checklist start time (on by default).
  • Shorten location name to the first punctuation mark — trims long location names at the first comma, dash, colon, @, or parenthesis. For example, "Central Park, Manhattan, NY" becomes "Central Park".
  • If photos have the same HH-MM-SS, add tenths (if available) — when two or more photos share an identical second-level timestamp, appends the tenths digit from the EXIF subsecond field to differentiate them.

The photo table

The table shows one row per photo matching the current filter. Each row has:

ColumnDescription
(checkbox)Check to include this photo in the rename; uncheck to skip it
Current FilenameThe file's existing name on disk
Proposed FilenameThe name Yearbirder will assign based on your format settings
StatusReadiness indicator (see below)

Hover over any row to see a thumbnail preview of that photo.

Status column values

StatusMeaning
ReadyThe proposed name is valid and the rename will proceed
No ChangeThe proposed name is identical to the current name; nothing will happen
Duplicate: Append _NMultiple photos would share the same proposed name; Yearbirder will append a numeric suffix (_1, _2, etc.) to each

No NameAll slots are set to (omit), so no name can be generated

Selecting and renaming

Use Select All and Select None to quickly check or uncheck every row. The count label shows how many photos are currently selected. When you are satisfied with the proposed names, click Rename. A confirmation dialog shows how many files will be renamed before any changes are made. After renaming, the Status column updates to Done for each successful file, or shows an error message (such as Permission denied or Name already in use) for any file that could not be renamed.

23. Species Gallery

Open with Photos → Species Gallery. Available once a photo catalog is loaded.

The Species Gallery displays one representative photo for each species in your collection — the highest-rated photo you have for that species. Photos are arranged in a responsive grid and sorted in taxonomic order, making it a visual life list of every species you have photographed.

The number of columns adjusts automatically as you resize or maximise the window, so the grid always makes full use of the available space.

Each tile shows:

  • A thumbnail of the best-rated photo for that species.
  • The species common name below the thumbnail.

Hovering over a tile highlights it. Click a tile to open a Photos window showing all of your photos of that species (subject to the current filter).

The window title and subtitle show how many species have at least one photo under the current filter. Like all Yearbirder windows, the Species Gallery responds to the Sighting Filter — for example, set a location filter to see a gallery of every species you have photographed at a particular site.

Slideshow and Show All buttons

Two buttons appear in the top-right corner of the Species Gallery header:

  • Slideshow — opens the Slideshow options dialog and then launches a full-screen slideshow of all photos matching the gallery's current filter. See Slideshow for details on the options dialog.
  • Show All — opens a Photos window showing every individual photo (not just one per species) that matches the gallery's current filter.

24. Individual Species Window

The Individual Species window opens when you click a species name in the Species List, or from the right-click menu in most list windows.

It shows a comprehensive view of your sightings of one particular species:

  • All sightings — every checklist on which you recorded this species, with date, location, and count.
  • Locations summary — a breakdown of sightings by location.
  • Yearly summary — how many times you recorded the species in each year.
  • Monthly summary — which months you most often record this species (useful for understanding migration timing).
  • Links — direct links to the species' page on Wikipedia and AllAboutBirds.org.
  • Photos — if you have photos of this species in your photo catalog, thumbnails are shown. Click a thumbnail to open the Enlargement view.

Photo thumbnail in the header

If you have photos of the species in your photo catalog, the highest-rated photo appears as a thumbnail in the window header, between the species data panes and the action buttons. The thumbnail scales to fit the available space without distortion. Click the thumbnail to open a Photos window showing all of your photos of that species.

25. Printing and PDF Export

Printing

Click Print on the toolbar or choose File → Print. The currently active window is sent to the printer. Most list and report windows print sensibly; map and photo windows print a screenshot of the current view.

PDF Export

Click Create PDF on the toolbar or choose File → Create PDF. Yearbirder exports the active window to a PDF file. A file save dialog lets you choose the destination. This is the recommended way to save or share a specific view of your data.

26. Tips and Tricks

Keep multiple windows open for comparison

Yearbirder allows many windows to be open at the same time. For example, open a Species List filtered to one location, then open a second Species List filtered to a different location, and use Windows → Tile Windows to view them side by side.

Seasonal migration analysis

Set the seasonal range filter to a narrow window (for example, May 1–May 15) and open the Date Totals report. You will see how your spring migration counts vary from year to year across those same two weeks.

Finding your best single day

Open Date Totals with no filters set and sort the Date Totals table by Species (descending). Your highest single-day species count rises to the top.

Tracking a family across geography

Set the Family filter to a family you are interested in (e.g., Shorebirds), then open the US States Choropleth. The shading immediately shows which states you have recorded shorebirds in.

Combining Name Search with other filters

The Name Search field accepts a partial name and filters the Species List in real time. This is the fastest way to find a species when you are not sure of its exact name — type "thrush" and all thrushes appear regardless of which family they belong to.

Using graphs to understand your birding patterns

  • Phenology Chart + species filter: select a favourite migrant species in the Species filter, then open the Phenology Chart. You will immediately see whether your records show the species arriving earlier or later in recent years.
  • Checklist Scatter + location filter: set a location filter and open the Checklist Scatter. Clusters of dots reveal your typical productivity at that site — and outlier dots (many species in a short time, or few species in a long session) are worth investigating.
  • Species Accumulation + location filter: filter to a county or location you visit regularly and open the Species Accumulation chart. Bars with a large bright-blue new-species segment were standout years for that location.
  • Cumulative Curve with no filter: open the Cumulative Species Curve with all filters cleared to see your all-time life-list growth curve. The slope flattens as you become a more experienced birder — steep jumps often correspond to trips to new regions.

Photo workflow tips

  • Rate your photos immediately after a field outing while the session is fresh. Use the Enlargement view to assign ratings of 1–5.
  • Back up your photo catalog file alongside your eBird data file. It is the only record of your photo associations.
  • Use the Photo Filter to find your best shots: set Rating Range to 4–5 and open the Photos gallery to see only your highest-rated photos.
  • Update EXIF data after any significant photo editing session or file reorganisation to keep the metadata in sync.

27. Saving Preferences

You can tell Yearbirder which folder you use to download ebird files. Yearbirder will load the most recent ebird file from that folder whenever Yearbirder starts.

You can also tell Yearbirder which photo catalog to open automatically when it starts.

Click Yearbirder → Preferences on the menu bar to set your preferences.
