1. Quick Start

This section gets you from a fresh install to your first results in five minutes.

Step 1 — Download your eBird data

With a browser such as Chrome, Safari, or Edge, go to https://ebird.org/downloadMyData, and choose Request My Observations. eBird will email you a link to a .zip file containing your complete sightings history. Open the email, click the link, and save the file somewhere easy to find. Yearbird can open the .zip file directly — there is no need to unzip it first. By default, most browsers will save the file in your Downloads folder.

Step 2 — Open your data in Yearbird

Launch Yearbird. Click the Open button on the toolbar (or choose File → Open). Navigate to the .zip file you downloaded and click Open. Yearbird loads your data and displays list of the species in the file.

Step 3 — Generate your first list

Browse through the options on the filter on the left side of the screen. Set a few filter options, like county or family, and then click the Species button on the toolbar. A new window opens showing the species you have recorded, sorted taxonomically, along with the first date, most recent date, and checklist count for each. You can click on the column headers to sort the columns. You can also right-click on any information in the list to send that information to the filter.

Step 4 — Explore further

  • Click Map to see all your sighting locations plotted on an interactive map.
  • Click Date Totals to see which years, months, and individual dates you recorded the most species.
  • Click Photos (after setting up a Photo Settings file — see Section 17) to browse your bird photos alongside their sighting data.

You are up and running. Everything else in this guide describes these features in detail.

2. The Main Window

Yearbird uses a Multiple Document Interface (MDI). The main window contains:

  • A menu bar across the top with menus for File, Filter, Lists, Map, Reports, Graphs, Photos, Maps, and Windows.
  • A toolbar just below the menu bar with quick-access buttons for the most common actions.
  • A Standard Filter panel docked on the left side.
  • A Photo Filter panel docked on the right side.
  • A large MDI area in the centre where all child windows (lists, reports, maps, photos) are displayed.
  • A status bar at the very bottom showing the current data file and record counts.

Any number of child windows can be open at the same time. Use Windows → Tile Windows or Windows → Cascade Windows to arrange them. Close individual windows with their close button, or close all at once with Windows → Close All Windows.

Each child window's title bar shows a descriptive label that reflects the active filter — for example, "Species: Warbler; New York; 2024 (42 Species)".

3. Loading Your eBird Data

Opening a data file

Choose File → Open or click the Open toolbar button. Yearbird accepts the standard eBird download CSV (the file is typically named something like MyEBirdData.csv).

Loading time depends on the size of your dataset. A typical dataset of 10,000–50,000 sightings loads in a few seconds. Once loaded, the status bar shows the total number of sightings and species.

Closing a data file

Choose File → Close to unload the current dataset. Any open child windows that depend on the data will close automatically.

What data is loaded

Yearbird reads the following fields from your eBird CSV:

  • Species common name and scientific name
  • Taxonomic order, family, and order
  • Observation date and time
  • Country, state/province, county, and location name
  • Location coordinates (latitude/longitude)
  • Checklist ID
  • Number of individuals counted

4. The Standard Filter

The Standard Filter panel is the primary way to focus every report and list on the data you care about. It is docked on the left side of the main window and always visible. All open child windows update immediately whenever you change any filter setting.

Location filters

The location filters are hierarchical — selecting a more specific level automatically constrains the broader levels. You can filter at any one of these levels:

FilterDescription
RegionA broad geographic region (e.g., North America)
CountryA specific country (e.g., United States)
StateA state or province
CountyA county or sub-region within a state
LocationA specific named eBird location

Select **All** in any drop-down to clear that level's filter.

Taxonomic filters

FilterDescription
OrderBird order (e.g., Passeriformes)
FamilyBird family (e.g., Parulidae – Wood-Warblers)
SpeciesA single species name
Name SearchFree-text search; matches any part of a species' common name

The taxonomic filters also follow a hierarchy for the purposes of the window title: if a Species is selected, Family and Order are not shown in the title (they are implied). If only a Family is selected, Order is not shown. Only the most specific level selected is displayed.

Date filters

FilterDescription
Date OptionsQuick presets: No date filter, This year, Last year, etc.
Start DateBeginning of a custom date range (yyyy-mm-dd)
End DateEnd of a custom date range (yyyy-mm-dd)

Seasonal range filters

Seasonal filters let you examine a recurring window within each year regardless of the year itself — useful for studying migration patterns.

FilterDescription
Seasonal Range OptionsQuick presets for common seasonal windows
Start Seasonal RangeMonth and day marking the start of the seasonal window
End Seasonal RangeMonth and day marking the end of the seasonal window

For example, setting a seasonal range of May 1 – May 31 shows all May sightings across all years.

Clearing filters

  • Click Clear all filters on the toolbar to reset every filter at once.
  • Individual filters can be cleared by selecting the **All** option in their drop-down.

5. The Photo Filter

The Photo Filter panel is hidden until you open a Photo Settings file (see Section 18), at which point it appears automatically on the right side of the main window. It filters photo-related results in addition to the Standard Filter — only photos that match both the Standard Filter and the Photo Filter are shown.

FilterDescription
Rating RangeMinimum and maximum photo rating (your personal quality score)
SpeciesShow only species that have photos / that have no photos
CameraFilter by specific camera model
LensFilter by specific lens model
Shutter Speed RangeMinimum and maximum shutter speed
Aperture RangeMinimum and maximum aperture (f-stop)

All Photo Filter fields work as range selectors. Select **All** to disable a particular filter.

6. Species List

Open with Lists → Species or the Species toolbar button.

The Species List shows every bird species (and other taxa) you have recorded that matches the current filter. Each row shows:

ColumnDescription
TaxTaxonomic sort order number
SpeciesCommon name (click to open the Individual Species window)
FirstDate of your first sighting of this species
LastDate of your most recent sighting
ChecklistsNumber of checklists on which this species appears
% of ChecklistsThe percentage of your total checklists that include this species

Species that are not full species (slash species, spuhs, hybrids) are listed separately and marked distinctly.

The window title shows the total count — for example, "Species: New York (42 Species)".

Clicking a species

Click any species name to open an Individual Species window (see Section 19) showing the full history of your sightings of that species.

Right-click menu

Right-clicking a row in the species table offers options to set the species as the active filter, open associated web pages, or open the individual species window.

7. Locations List

Open with Lists → Locations or the Locations toolbar button.

Shows all locations where you have recorded sightings under the current filter:

ColumnDescription

FirstDate of your first visit
LastDate of your most recent visit

The count of locations is shown in the window title, e.g., "Locations: All species, locations, and dates (702 Locations)".

Right-clicking a location lets you set it as the active location filter, or open it on a map.

8. Checklists List

Open with Lists → Checklists or the Checklists toolbar button.

Shows every checklist that matches the current filter:

ColumnDescription

9. Families Window

Open with Lists → Families or the Families toolbar button.

The Families window has two panes:

  • Left pane: A scrolling list of every bird family represented in your sightings under the current filter.
  • Right pane: The species from the selected family, along with sighting statistics.

Click a family name in the left pane to populate the right pane. This is an efficient way to browse your sightings by taxonomic family — for example, quickly seeing all your warbler or shorebird records.

The window title shows the number of families, e.g., "Families: New York (12 Families)".

10. Date Totals

Open with Reports → Date Totals or the Date Totals toolbar button.

The Date Totals window presents three side-by-side tables:

Year Totals

One row per year, showing how many species and checklists you recorded in that year.

Month Totals

One row per calendar month (January through December), showing totals across all years.

Date Totals

One row per individual date on which you submitted a checklist, showing species and checklist counts for that date.

All three tables are sortable by any column. This report is useful for identifying your most productive years, your best months for birding, and your record single-day species counts.

The window title shows the total number of unique dates, e.g., "Date Totals: All species, locations, and dates (365 Dates)".

11. Location Totals

Open with Reports → Location Totals or the Location Totals toolbar button.

Location Totals presents five tables on separate tabs:

TabDescription
Region TotalsSpecies counts by broad geographic region
Country TotalsSpecies counts by country
State TotalsSpecies counts by state/province
County TotalsSpecies counts by county
Location TotalsSpecies counts by individual named location

Each table shows rank, name, species count, and checklist count. Sorted by rank (most species first) by default.

12. Big Report

Open with Reports → Big Report or the Big Report toolbar button.

The Big Report is the most comprehensive single report in Yearbird. It combines multiple dimensions of your data into one window with multiple tabs:

  • Species — Full species list with counts, first and last dates.
  • Dates — Date-by-date breakdown of species seen.
  • Locations — Location-by-location breakdown.
  • New Species — Species seen for the first time at each location.
  • Checklists — All checklists in the filtered set.

The window title includes a species count, e.g., "Big Report: New York (87 Species)".

13. Compare Lists

Open with Reports → Compare Lists or by clicking the Compare Lists button in the toolbar.

Compare Lists lets you select two separate lists and compare their species. This is useful for questions like:

  • "Which species did I see in 2023 that I did not see in 2024?"
  • "Which species have I seen at location A but not at location B?"

To compare lists:

  1. Use the main window's filter and the Species button on the toolbar to create at least two list windows.
  2. Click the Compare Lists button on the toolbar to open the Compare Lists window.
  3. Select one list in the left-hand dropdown box and select another list in the right-hand dropdown list.
  4. click the Compare button in the middle to see side-by-side results.

14. Graphs

Open graphs from the Graphs menu. Eleven chart types are available, each responding to the current Standard Filter. All graphs display a hover tooltip when you move the cursor over a data point or bar — the tooltip shows relevant details such as date, species count, or location. Clicking on most data points opens a Species List window filtered to that point's context.

Bar Graph

Open with Graphs → Bar Graph.

Shows species counts aggregated over time, with four granularity options selectable via radio buttons at the top of the window:

  • By Year — one bar per year; the default when your data spans multiple years.
  • By Month — one bar per calendar month, aggregated across all years. Shows which months are most productive regardless of year.
  • By Month-Year — one bar per calendar month within each year (e.g., Jan-2022, Feb-2022, …). Useful for fine-grained temporal analysis. Disabled if there are more than 100 month-year combinations in the filtered data.
  • By Day — one bar per individual date. Useful when the filter covers a narrow date range. Disabled if there are more than 100 days in the filtered data.

Click any bar to open a Species List for that time period.

Cumulative Species Curve

Open with Graphs → Cumulative Species Curve.

A line chart showing how your total species count has grown over time as each new species was first recorded. The x-axis is date and the y-axis is the running cumulative total. The final point shows the total number of species seen under the current filter.

This chart is most interesting with no date filter set — it shows your life-list growth curve from your earliest eBird records to the present. With a location filter applied, it shows how quickly you added new species at that location.

Hover over the line to see the cumulative count and date at any point. Click to open a Species List of all species seen up to and including that date.

Species Heatmap

Open with Graphs → Species Heatmap.

A grid showing species counts for every month of every year in your data. Rows are years (most recent at the top) and columns are months (January through December). Cell colour ranges from light blue (low count) to dark blue (high count); cells with no data are shown in grey. When the grid is 15 years or fewer, each cell is also labelled with its species count.

The heatmap immediately reveals seasonal patterns — which months of each year were most productive — and gaps in your birding record.

Hover over any cell to see the month, year, and species count. Click a cell to open a Species List for that month and year.

Species Accumulation

Open with Graphs → Species Accumulation.

A stacked bar chart showing, for each year, how many species you recorded that were new to you under the current filter (bright blue, labelled with a + count) versus how many were species you had already seen in a previous year (muted blue). Together the two segments show your total species count for that year.

This graph answers questions such as: "How many new species did I add to my county list in 2023?" or "Has my backyard species accumulation slowed down in recent years?"

Hover over a bar to see the year, total count, and new-species count. Click a bar to open a Species List for that year.

Top Locations

Open with Graphs → Top Locations.

A horizontal bar chart showing your top locations by species count (up to 20 locations). Each bar is split into two segments:

  • Muted blue — species you had already recorded elsewhere before visiting this location.
  • Bright blue — species you recorded at this location for the first time ever in your database (life birds).

The total species count is labelled at the right end of each bar. This chart shows not only which locations have been most productive, but which locations have contributed the most unique species to your life list.

Hover over a bar to see the full location name, total species count, and life bird count. Click a bar to open a Species List filtered to that location.

Checklist Scatter

Open with Graphs → Checklist Scatter.

A scatter plot showing every checklist matching the current filter as a dot, with duration in minutes on the x-axis and species count on the y-axis. Dots are coloured by season:

ColourSeasonMonths
Steel blueWinterDecember 20 – March 19
GreenSpringMarch 20 – June 19
GoldenSummerJune 20 – September 19
OrangeFallSeptember 20 – December 19

Incidental observations (checklists with no recorded duration) are excluded; a note at the bottom right of the chart shows how many were omitted.

Important: the species count on the y-axis reflects only species matching the active filter, not all species on the checklist. For example, if you filter by Tyrannidae, a checklist on which you recorded 3 flycatchers out of 40 total species will appear at y = 3.

Hover over a dot to see the date, location, duration, and species count. Click a dot to open a Species List for that checklist.

Phenology Chart

Open with Graphs → Phenology Chart.

A dot chart designed to reveal the seasonal timing of species occurrence. Each dot represents one date on which a species (or species matching the filter) was recorded, plotted by day of year on the x-axis and year on the y-axis. Dots are coloured by season using the same palette as the Checklist Scatter. If the same species was recorded at multiple locations on the same day, only one dot is plotted for that date (the tooltip lists all the locations).

This chart is most powerful when a species, family, or order filter is set. For a single species it immediately shows:

  • Typical arrival and departure dates
  • Whether the species is a year-round resident or a seasonal visitor
  • Whether the timing has shifted over the years
  • Unusually early or late individual records

If no species, family, or order filter is set, Yearbird will warn you before generating the chart, since plotting all species together produces a very cluttered result.

Hover over a dot to see the exact date and the location or locations seen that day. Click a dot to open a Species List for that date.

First of Year Chart

Open with Graphs → First of Year Chart.

A dot chart plotting the first sighting of each species in each year. The x-axis shows day of year (January through December) and the y-axis shows year. Each dot represents one or more species whose first record of the year fell on the same date; hovering reveals the species list for that dot. Dots are coloured by season using the same palette as the Checklist Scatter.

This chart is ideal for tracking arrival timing — when do your warblers return each spring? Has arrival crept earlier over the years? Apply a Family or species filter before opening the chart for the most focused view.

Hover over a dot to see the date and the species recorded for the first time that year. Click a dot to open a Species List for that date.

Last of Year Chart

Open with Graphs → Last of Year Chart.

The mirror image of the First of Year Chart: plots the last sighting of each species in each year, using the same axes and colour scheme. Useful for tracking departure timing and lingering late records.

Hover over a dot to see the date and the species recorded for the last time that year. Click a dot to open a Species List for that date.

Pie Charts

Open with Graphs → Pie Charts. Two pie chart variants are available, selectable via radio buttons at the top of the window:

  • Pie Chart by Species — each wedge represents a bird family (or order), sized by the number of species you have recorded in it under the current filter.
  • Pie Chart by Individual Tallies — each wedge represents a family (or order), sized by the total number of individual birds tallied. Unknown counts ("X" in eBird) are counted as 1.

Toggle between Families and Orders using the radio buttons below the chart title. Wedges smaller than 0.5% of the total are unlabelled to avoid clutter; percentage labels appear only on wedges of 2% or larger.

Hover over a wedge to see the family or order name, its count, and the species it contains. Click a wedge to open a Species List filtered to that family or order.

15. Maps

Yearbird's Maps menu offers a range of interactive map types. All maps are pan-and-zoom capable using your mouse or trackpad, respond to the Standard Filter, and open in MDI child windows like any other report. Hovering over a marker or bubble shows a tooltip; clicking most markers opens the Location window for that site.

Location Map

Open with the Map toolbar button or Map → Map.

Plots every location where you have recorded sightings under the current filter as a circle marker, sized by species count. Hover over a marker to see the location name and species list. Click a marker to open the Location window for that site.

Animated Lifer Map

Open with Maps → Animated Lifer Map.

Animates your life list building geographically over time. Each dot represents a lifer — the first time you recorded a species anywhere in your database — placed at the sighting's GPS coordinates and revealed in chronological order. Dots are coloured on a cream-to-orange gradient from earliest to most recent lifer; the newest dot flashes blue before settling to its gradient colour when the next dot appears.

A control bar at the bottom of the map provides:

  • ⏮ Restart — jump back to the beginning.
  • ▶ / ⏸ Play/Pause — start or pause the animation.
  • Scrubber — drag to jump to any point in the sequence.
  • Speed slider (🐢 – 🐇) — adjust animation speed from leisurely to rapid.
  • Lifer info line — shows the species name, location, and date of the most recently revealed dot.

Hover over any dot to see its species name, lifer number, date, and location. Click a dot to open a popup with the same information.

Effort Map by Time

Open with Maps → Effort Map by Time.

A bubble map where each bubble represents a checklist location. Bubble area is proportional to the total cumulative time you have spent birding at that location (in minutes, summed across all matching checklists). Locations where you have invested the most time appear as the largest bubbles.

Hover over a bubble to see the location name, total time, and checklist count. If some checklists at a location have no recorded duration, the tooltip notes how many are untimed. Click a bubble to open the Location window for that site.

Effort Map by Checklists

Open with Maps → Effort Map by Checklists.

The same bubble map concept, but bubble area is proportional to the number of checklists submitted at each location rather than total time. Useful for seeing where you bird most frequently regardless of duration.

Hover over a bubble to see the location name, checklist count, and total time. Click a bubble to open the Location window for that site.

Species Total Map

Open with Maps → Species Total Map.

A bubble map where bubble area is proportional to the number of species recorded at each location under the current filter. Slashes, spuhs, and hybrids are excluded from the species count using the same rules as the Species List. Apply a taxonomic filter to compare how many warblers — or any other group — you have found at each of your locations.

Hover over a bubble to see the location name and species count. Click a bubble to open the Location window for that site.

Individuals Total Map

Open with Maps → Individuals Total Map.

A bubble map where bubble area is proportional to the total number of individual birds tallied at each location. Sightings recorded with an unknown count ("X" in eBird) contribute 1 to the total. The tooltip notes how many checklists contain at least one uncounted species.

Hover over a bubble to see the location name, individual count, and any caveat about uncounted entries. Click a bubble to open the Location window for that site.

Geolocated Photos

Open with Maps → Geolocated Photos. Available only when a Photo Settings file is loaded.

Plots your photos on a clustered marker map. Each marker represents one photo; clicking it opens a popup showing a thumbnail of the photo, the species name, date, and location. Markers cluster together at lower zoom levels and spread apart as you zoom in. Multiple photos taken at exactly the same GPS coordinates are automatically spread into a Fibonacci spiral so they remain individually clickable.

16. Choropleth Maps

Choropleth maps are accessed via Maps → Choropleths. Six geographic regions are available, each with two modes selectable from the By Species and By Checklists submenus.

All choropleth maps respond to the Standard Filter — for example, set a Family filter to see which states you have recorded warblers in. Clicking a shaded region opens a Species List (By Species mode) or a Checklists List (By Checklists mode) filtered to that region.

By Species mode

Each region is shaded according to the number of distinct species you have recorded there under the current filter. Darker shading = more species. Hover over a region to see its name, species count, and a list of the most recently recorded species.

By Checklists mode

Each region is shaded according to the number of checklists you have submitted there. Hover over a region to see its name, checklist count, and the dates of your most recent visits (newest first).

17. Photos

Yearbird can associate your personal bird photos with your eBird sightings. This unlocks a photo gallery view, displaying photos that match the filter, and filtering by camera, lens, and exposure settings using the photo filter that will appear on the right of the screen.

Overview of the photo system

  • Your photos remain in their original folders on your computer. Yearbird stores references to them, not copies.
  • Photo associations and metadata are stored in a Photo Settings file (a CSV file you create and save using Yearbird). See Section 18.
  • Yearbird reads EXIF data from each JPEG file automatically: camera model, lens, shutter speed, aperture, focal length, and ISO.

Opening the Photos window

Once a Photo Settings file is loaded, click Photos on the toolbar or choose Lists → Photos. The Photos gallery opens, showing thumbnails of all photos that match the current filter.

The Photos gallery

Photos are arranged in a grid. Each thumbnail shows:

  • The photo itself
  • The species name
  • The date the photo was taken
  • Your rating for the photo (if set)

Click any thumbnail to open the Enlargement view for that photo.

Sorting photos

You can sort the gallery by:

  • Taxonomy — groups photos by species in taxonomic order
  • Date — most recent first or oldest first
  • Rating — highest rated first
  • Species name — alphabetical

Photo Filter

Use the Photo Filter panel (right side of the main window) to narrow the gallery by camera, lens, exposure settings, or rating. See Section 5 for the full list of filter options.

Enlargement view

Click any thumbnail to open the Enlargement view, which shows the full-resolution photo with pan and zoom. From the Enlargement view you can:

  • Navigate between photos (previous/next arrows).
  • Set or change the rating for the photo.
  • View EXIF data — camera, lens, shutter speed, aperture, focal length, ISO.
  • Detach a photo from its sighting (removes the association without deleting the file).
  • Delete a photo reference entirely from the Photo Settings.

18. Adding Photos to the Photo Settings File

Yearbird uses a Photo Settings file to associate photos on your hard drive with your ebird sightings. Associating photos with your ebird sightings lets you view photos that match the main window's filter. For example, Yearbird can show you all the photos you've taken of a species, or the photos from a certain day or from a location. Yearbird stores the photo file names, dates, times, species names, and camera information in a separate data file. Think of this file as your photo library within Yearbird.

Adding Photos To Yearbird

  1. Choose Photos → Add Photos.
  2. A file dialog opens. Navigate to the folder containing your bird photos and select one or more JPEG files.
  3. For each photo, Yearbird asks you to identify the associated bird species and sighting date. If the correct sighting location and date does not appear as an option, make sure that you have logged that sighting in ebird itself; you cannot add a photo to Yearbird unless that sighting is also in your eBird data.
  4. Once you have associated photos, choose Photos → Save Photo Settings to save your Photo Settings file. Remember where youy save this file, as you will want to open it in the future to view your photos with Yearbird.

Opening an existing Photo Settings file

Choose Photos → Open Photo Settings File and navigate to your saved .csv file. Yearbird loads all photo associations and EXIF data. The Photo Filter panel becomes active.

Saving changes

After adding, editing, or removing photo associations, choose Photos → Save Photo Settings to write changes back to the file. Yearbird will warn you if you try to close the file with unsaved changes.

Closing the Photo Settings file

Choose Photos → Close Photo Settings File. This unloads all photo data from memory. The Photo Filter panel becomes inactive.

Adding more photos

With a Photo Settings file already loaded, choose Photos → Add Photos to import additional photos. The new associations are merged into the existing file; save afterwards.

Editing photos by filter

Choose Photos → Edit Photos by Filter to make batch changes — for example, to update the rating of all photos from a particular outing, or to reassign a group of photos to a different species.

Updating EXIF data

If you have moved or re-exported your photo files, choose Photos → Update EXIF Data for All Photos. Yearbird re-reads the EXIF metadata from every file referenced in the Photo Settings and updates its records. This is also useful if you use photo editing software that writes new EXIF tags.

Cleaning up missing references

If photos have been moved or deleted from your hard drive, Yearbird may show broken references. Use Photos → Remove Missing Photo References to scan for and remove any associations pointing to files that no longer exist.

What the Photo Settings file contains

The CSV file stores one row per photo reference, with columns for:

  • File path to the JPEG
  • Associated species common name
  • Associated sighting date
  • Your personal rating
  • Camera model (from EXIF)
  • Lens model (from EXIF)
  • Shutter speed (from EXIF)
  • Aperture (from EXIF)
  • Focal length (from EXIF)
  • ISO (from EXIF)

You can open the CSV in a spreadsheet application to inspect or bulk-edit values, but be careful to preserve the column order and formatting.

19. Individual Species Window

The Individual Species window opens when you click a species name in the Species List, or from the right-click menu in most list windows.

It shows a comprehensive view of your sightings of one particular species:

  • All sightings — every checklist on which you recorded this species, with date, location, and count.
  • Locations summary — a breakdown of sightings by location.
  • Yearly summary — how many times you recorded the species in each year.
  • Monthly summary — which months you most often record this species (useful for understanding migration timing).
  • Links — direct links to the species' page on Wikipedia and AllAboutBirds.org.
  • Photos — if you have photos of this species in your Photo Settings, thumbnails are shown. Click a thumbnail to open the Enlargement view.

20. Printing and PDF Export

Printing

Click Print on the toolbar or choose File → Print. The current active child window is sent to the printer. Most list and report windows print sensibly; map and photo windows print a screenshot of the current view.

PDF Export

Click Create PDF on the toolbar or choose File → Create PDF. Yearbird exports the active child window to a PDF file. A file save dialog lets you choose the destination. This is the recommended way to save or share a specific view of your data.

21. Tips and Tricks

Keep multiple windows open for comparison

Because Yearbird is an MDI application, you can open several child windows simultaneously. For example, open a Species List filtered to one location, then open a second Species List filtered to a different location, and use Windows → Tile Windows to view them side by side.

Seasonal migration analysis

Set the seasonal range filter to a narrow window (for example, May 1–May 15) and open the Date Totals report. You will see how your spring migration counts vary from year to year across those same two weeks.

Finding your best single day

Open Date Totals with no filters set and sort the Date Totals table by Species (descending). Your highest single-day species count rises to the top.

Tracking a family across geography

Set the Family filter to a family you are interested in (e.g., Shorebirds), then open the US States Choropleth. The shading immediately shows which states you have recorded shorebirds in.

Combining Name Search with other filters

The Name Search field accepts a partial name and filters the Species List in real time. This is the fastest way to find a species when you are not sure of its exact name — type "thrush" and all thrushes appear regardless of which family they belong to.

Using graphs to understand your birding patterns

  • Phenology Chart + species filter: select a favourite migrant species in the Species filter, then open the Phenology Chart. You will immediately see whether your records show the species arriving earlier or later in recent years.
  • Checklist Scatter + location filter: set a location filter and open the Checklist Scatter. Clusters of dots reveal your typical productivity at that site — and outlier dots (many species in a short time, or few species in a long session) are worth investigating.
  • Species Accumulation + location filter: filter to a county or location you visit regularly and open the Species Accumulation chart. Bars with a large bright-blue new-species segment were standout years for that location.
  • Cumulative Curve with no filter: open the Cumulative Species Curve with all filters cleared to see your all-time life-list growth curve. The slope flattens as you become a more experienced birder — steep jumps often correspond to trips to new regions.

Photo workflow tips

  • Rate your photos immediately after a field outing while the session is fresh. Use the Enlargement view to assign ratings of 1–5.
  • Back up your Photo Settings CSV alongside your eBird data file. It is the only record of your photo associations.
  • Use the Photo Filter to find your best shots: set Rating Range to 4–5 and open the Photos gallery to see only your highest-rated photos.
  • Update EXIF data after any significant photo editing session or file reorganisation to keep the metadata in sync.

22. Saving Preferences

You can tell Yearbird which folder you use to download ebird files. Yearbird will load the most recent ebird file from that folder whenever Yearbird starts.

You can also tell Yearbird which Photo Settings file to use automatically when it starts.

Click Yearbird → Preferences on the menu bar to set your preferences.
